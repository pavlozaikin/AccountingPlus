# AccountingPlus

Веб-застосунок «Облік+», реалізований за допомогою Django. Забезпечує авторизацію та роботу зі списком осіб, що перебувають на військовому обліку. Форми інтегровані з майбутньою експертною системою на базі бібліотеки Experta.

## Інструкція для першого запуску на macOS

Ця інструкція допоможе вам запустити проєкт на вашому комп'ютері Mac, навіть якщо у вас немає технічних навичок. Просто дотримуйтесь кроків, наведених нижче.

### Крок 1: Встановлення інструментів розробника

Для роботи програми потрібні спеціальні інструменти. Вони встановлюються однією простою командою.

1.  **Відкрийте термінал.**
    *   Натисніть `Cmd + Пробіл`, щоб відкрити пошук Spotlight.
    *   Введіть `Термінал` і натисніть `Enter`.

2.  **Встановіть Command Line Tools.**
    *   Скопіюйте та вставте в термінал команду нижче, після чого натисніть `Enter`.

    ```bash
    xcode-select --install
    ```
    *   З'явиться вікно з пропозицією встановити інструменти. Натисніть "Install" і дочекайтеся завершення процесу.

### Крок 2: Встановлення Homebrew

Homebrew — це менеджер пакунків, який спрощує встановлення програмного забезпечення.

1.  **Встановіть Homebrew.**
    *   Вставте в термінал команду нижче та натисніть `Enter`.

    ```bash
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    ```
    *   Термінал попросить вас ввести пароль від вашого комп'ютера. Вводьте його (символи не будуть відображатися) і натисніть `Enter`.
    *   Процес встановлення може зайняти кілька хвилин.

### Крок 3: Встановлення Python

Тепер встановимо мову програмування Python.

1.  **Встановіть Python.**
    *   Виконайте в терміналі наступну команду:

    ```bash
    brew install python
    ```

### Крок 4: Завантаження коду проєкту

1.  **Завантажте проєкт у вигляді ZIP-архіву.**
    *   На сторінці, де розміщено код (наприклад, GitHub), знайдіть зелену кнопку `<> Code` і виберіть у меню `Download ZIP`.
    *   Збережіть архів у стандартну папку `Завантаження` (Downloads).

2.  **Розпакуйте архів.**
    *   Відкрийте Finder, перейдіть до папки `Завантаження`.
    *   Знайдіть завантажений файл (наприклад, `accounting-plus-main.zip`) і двічі клацніть по ньому. Поруч з'явиться папка з кодом проєкту (наприклад, `accounting-plus-main`).

3.  **Перейдіть до папки проєкту в терміналі.**
    *   Для цього введіть команду `cd` і перетягніть папку проєкту з Finder прямо у вікно термінала. Команда має виглядати приблизно так:
    ```bash
    cd ~/Downloads/accounting-plus-main
    ```
    *   Натисніть `Enter`.

### Крок 5: Налаштування та запуск

1.  **Створіть віртуальне середовище.**
    *   Це ізольований простір для залежностей проєкту.

    ```bash
    python3 -m venv venv
    ```

2.  **Активуйте середовище.**

    ```bash
    source venv/bin/activate
    ```
    *   Ви побачите `(venv)` на початку рядка в терміналі.

3.  **Встановіть залежності.**

    ```bash
    pip install -r requirements.txt
    ```

4.  **Застосуйте міграції бази даних.**
    *   Ця команда створить файл бази даних та підготує її до роботи.

    ```bash
    python manage.py migrate
    ```

5.  **Створіть адміністратора.**
    *   Цей користувач матиме доступ до адмін-панелі.

    ```bash
    python manage.py createsuperuser
    ```
    *   Введіть логін, email (необов'язково) та пароль.

6.  **Запустіть сервер.**

    ```bash
    python manage.py runserver
    ```

### Крок 6: Перевірка роботи

1.  **Відкрийте сайт у браузері.**
    *   Перейдіть за адресою `http://127.0.0.1:8000/`.
    *   Вас буде перенаправлено на сторінку входу. Використовуйте логін та пароль, які ви створили на попередньому кроці.

2.  **Як зупинити сервер.**
    *   Поверніться до термінала і натисніть `Ctrl + C`.

## Інформація для розробників

### Вимоги

- Python 3.11+
- Встановлені залежності з `requirements.txt`

### Структура

- `accountingplus/` — конфігурація Django-проєкту.
- `persons/` — застосунок для керування записами.
- `templates/` — базові шаблони, включно зі сторінкою авторизації.
- `static/` — стилі та скрипти, адаптовані під дизайн макетів.
- `assets/` — додаткові ресурси Bootstrap, доступні як статичні файли.

### Експертна система

У модулі `persons/recommendations.py` розміщено каркас експертної системи на Experta. Кнопка «Отримати рекомендацію» у формі надсилає дані до рушія та показує список порад без збереження змін у базі.

### Тестування адаптивності

Бічне меню працює як offcanvas-навігація на малих екранах (кнопка у шапці), а на великих — залишається фіксованою лівою панеллю.
