# AccountingPlus

Веб-застосунок «Облік+», реалізований за допомогою Django. Забезпечує авторизацію та роботу зі списком осіб, що перебувають на військовому обліку. Форми інтегровані з майбутньою експертною системою на базі бібліотеки Experta.

## Вимоги

- Python 3.11+
- Встановлені залежності з `requirements.txt`

```bash
pip install -r requirements.txt
```

## Перший запуск

1. Застосуйте міграції бази даних:

   ```bash
   python manage.py migrate
   ```

2. Створіть адміністратора (необов'язково, але корисно для доступу до Django admin):

   ```bash
   python manage.py createsuperuser
   ```

3. Запустіть сервер розробки:

   ```bash
   python manage.py runserver
   ```

4. Відкрийте `http://127.0.0.1:8000/` у браузері. Ви будете перенаправлені на сторінку входу (`/accounts/login/`). Після успішної авторизації відкривається список осіб на обліку.

## Структура

- `accountingplus/` — конфігурація Django-проєкту.
- `persons/` — застосунок для керування записами.
- `templates/` — базові шаблони, включно зі сторінкою авторизації.
- `static/` — стилі та скрипти, адаптовані під дизайн макетів.
- `assets/` — додаткові ресурси Bootstrap, доступні як статичні файли.

## Експертна система

У модулі `persons/recommendations.py` розміщено каркас експертної системи на Experta. Кнопка «Отримати рекомендацію» у формі надсилає дані до рушія та показує список порад без збереження змін у базі.

## Тестування адаптивності

Бічне меню працює як offcanvas-навігація на малих екранах (кнопка у шапці), а на великих — залишається фіксованою лівою панеллю.
